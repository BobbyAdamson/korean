  - I don't eat
  - 먹다 : 안 + 먹어요-> 안 먹어요.  : 안+ informal polite form
  - 먹+ 지 않아요[아나요] -> 먹지 않아요. : base form +지 않아요.

  - 가방이 비싸(요. 그래)서 안 사요 -> 가방이 비싸서 안 사요.
