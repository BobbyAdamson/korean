연습 1  ‘느라(고)’로 문장을 만들어 보세요. 
1. 사다 -> 새 차를 사느라 모은 돈(money save up)을 다 썼어요.
2. 쓰다 -> 크리스마스 카드를 쓰느라 정신이 없었어요.
3. 찾다 -> 집을 찾느라 힘드셨지요?
4. 돕다 -> 동생을 도느라 쉴 시간이 없었어요.
5. 퇴근하다 -> 급하게 퇴근하느라 회사에 스마트폰을 놓고 왔어요. 
6. 놀다 -> 친구와 노느라 선생님이 오셨지만 몰랐어요.
